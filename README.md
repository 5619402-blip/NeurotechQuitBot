# NeuroTech Quit Bot

Telegram-бот для поддержки пользователей в процессе отказа от зависимостей.

## Запуск

1. Установи зависимости:
   ```bash
   npm install
   ```

2. Скопируй `.env.example` в `.env` и заполни переменные:
   ```bash
   cp .env.example .env
   ```

3. Запусти бота:
   ```bash
   npm start
   ```

   Для разработки (с авто-перезагрузкой):
   ```bash
   npm run dev
   ```

## Структура проекта

```
NeurotechQuitBot/
├── src/
│   ├── index.js              # Точка входа
│   ├── config.js             # Конфигурация (переменные окружения)
│   ├── bot/
│   │   ├── bot.js            # Экземпляр Telegraf
│   │   ├── handlers/         # Обработчики команд и callback
│   │   └── keyboards/        # Клавиатуры и кнопки
│   ├── db/
│   │   └── schema.js         # Схема базы данных (таблицы и ENUM)
│   ├── services/
│   │   ├── payments/         # Платёжный провайдер
│   │   ├── player/           # Интеграция с защищённым плеером
│   │   ├── reminders/        # Напоминания (node-cron)
│   │   ├── storage/          # Хранилище аудиофайлов
│   │   └── gifts/            # Подарочные токены
│   └── utils/                # Вспомогательные утилиты
├── docs/                     # Документация (ТЗ)
├── memory/                   # Память проекта
├── .env.example              # Шаблон переменных окружения
└── package.json
```

## Зависимости

| Пакет | Назначение |
|---|---|
| `telegraf` | Telegram Bot API — middleware-архитектура, сцены, сессии |
| `dotenv` | Загрузка переменных окружения из `.env` |
| `node-cron` | Планировщик для напоминаний (48ч / 24ч / 7 дней) |
